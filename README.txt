== Description ==

Easily add `<meta name="title" content"YOUR SEO title content">` and `<meta name="description" content"YOUR SEO description content">` on WooCommerce Product Category pages.

- Light-weight simple implemenation
- Edit SEO attributes directly on WooCommerce Product Category pages
- Adds / Edits SEO meta "title" and "description" tags
- It's simple and easy to use.

`
<meta name="title" content="YOUR SEO title content" />
<meta name="description" content="YOUR SEO title description" />
`

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/simple-pcme` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= Where do I add or edit the meta tags? =

Go to Products > Categories. Click *edit* on the category you wish to add meta tags to. On the edit product category page, notice two new fields: SEO meta title, and SEO meta description.

= Can I add any more meta tags? =

No. At this time we only support adding and editing the meta tags with names "title", and "description". However, if you require additional functionality, please request it from the author.
